# Algorithm & Data Structures Repository

## 概要

このリポジトリは、競技プログラミングと技術面接の準備のために設計された、包括的なアルゴリズムソリューションとデータ構造実装のコレクションを提供します。コードベースは、正確性とパフォーマンス効率の両方を重視しながら、複数のプログラミング言語にわたって最適化された実装で基本的なコンピュータサイエンスアルゴリズムをカバーしています。

## 特徴

- **技法別整理**: プログラミング言語別ではなく、アルゴリズム技法別にソリューションを整理
- **多言語対応**: TypeScript、JavaScript、Python、Go、PHPでの実装
- **詳細な解析**: 各実装には複雑性解析、段階的説明、パフォーマンスベンチマークが含まれる
- **教育的価値**: アルゴリズムパターンとその様々な言語固有の最適化を学習可能

## 主要アルゴリズム

### 二分探索実装

**特徴:**
- ソート済み配列での効率的な検索
- O(log n)時間複雑度
- 回転配列や重複要素への対応

**実装例:**
- 2つのソート済み配列の中央値
- 回転配列での検索
- 要素の最初と最後の位置の検索

### データ構造システム

**Binary Indexed Tree (BIT):**
- 範囲和クエリ: O(log n)
- 単一要素更新: O(log n)
- 空間計算量: O(n)

**主要操作:**
- `update(index, value)`: 指定インデックスの値を更新
- `query(index)`: インデックス1からindexまでの累積和を取得
- `range_query(left, right)`: 指定範囲の和を計算
